Sopa de letras interactiva

  • Este proyecto contiene una sopa de letras la cual cuenta con un sistema de puntuación de última generación, temporizador inteligente y por último pero no menos importante incrementadores y reductores de tamaño.

  • Este proyecto fue creado mediante funcionalidades del popular lenguaje de marcado HTML, combinado con lenguajes de programación como JavaScript y Python
