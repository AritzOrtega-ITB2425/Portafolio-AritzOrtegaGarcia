Puzzle interactivo y personalizable

  • En este proyecto podrás jugar a hacer un puzzle mediante el número de piezas que tú quieras, este proyecto cuenta con funcionalidades exclusivas como poder implementar la imagen que tú quieras para realizar el puzzle con esa imagen, un botón de mezclado, temporizador inteligente y mucho más.

  • El proyecto fue creado con el popular lenguaje de marcas “HTML” y múltiples lenguajes de programación como Python y JavaScript
